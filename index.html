สถานะ CG รายเดือน

CG ผู้สูงอายุ เยี่ยมแล้ว ความคืบหน้า สถานะตามเกณฑ์การเยี่ยม

📂 ดูข้อมูลที่บันทึก

1. เลือกผู้ดูแล (CG) และผู้สูงอายุ

2. เดือน/ปี และจำนวนวันที่เยี่ยม

กรุณาเลือก CG และผู้สูงอายุก่อน

3. การวัดสัญญาณชีพ (Vital signs)

4. การประเมินสภาพและการดูแล

5. อัปโหลดรูปภาพ

รายชื่อผู้ดูแล (CG)

รหัส ชื่อ เลขบัตรประชาชน เบอร์โทร หมู่ ผู้สูงอายุ
รหัส ชื่อ อายุ CG คะแนน ADL กลุ่ม ADL เยี่ยม/เดือน โหมด สถานะ

🛠 การจัดการอุปกรณ์ (ครุภัณฑ์การแพทย์ / วัสดุการแพทย์)

บันทึกรายการครุภัณฑ์การแพทย์หรือวัสดุการแพทย์ที่สนับสนุนผู้สูงอายุแต่ละคน ตั้งแต่วันแรกที่ CG เข้าไปดูแล จนสิ้นสุดการดูแล

ชื่อผู้สูงอายุ เบอร์โทรญาติ เลขครุภัณฑ์ หมวดหมู่ ประเภทครุภัณฑ์ วันที่ได้รับ จำนวน สถานะการคืน

📄 ออกรายงานทะเบียนคุมครุภัณฑ์

รายงานนี้แยกต่างหากจากรายงานการเยี่ยมบ้านของ CG จะออกเป็นทะเบียนคุมรายการครุภัณฑ์ทั้งหมด (หรือกรองตามผู้สูงอายุที่เลือกด้านบน)

ออกรายงานการเยี่ยมบ้าน

1 หน้ากระดาษ A4 จะรวมการเยี่ยมของผู้สูงอายุคนเดียวกันสูงสุด 4 ครั้ง พร้อมสรุปรายงาน และช่องลงชื่อรับรองท้ายหน้า

รายงานข้อมูลผู้สูงอายุ (สถานะทั้งหมด)

รายงานรายชื่อผู้สูงอายุพร้อมข้อมูล ADL, CG ผู้ดูแล, ความคืบหน้าการเยี่ยมเทียบเป้าหมายรายเดือน, และสถานะการดูแล (ดูแลอยู่ / จำหน่ายแล้ว) ออกได้ทั้งไฟล์ Excel และ PDF

ประวัติผู้สูงอายุและผู้มีภาวะพึ่งพิงที่เคยได้รับการดูแล

รายชื่อผู้ที่จำหน่ายออกจากระบบเนื่องจากหายป่วย, อาการดีขึ้นไม่เข้าเกณฑ์การดูแล, เสียชีวิต หรือย้ายที่อยู่ พร้อมประวัติการเยี่ยมทั้งหมด — มองเห็นและจัดการได้เฉพาะ Admin และ CM

รหัสชื่ออายุCG เดิมเหตุผลวันที่จำหน่ายหมายเหตุ

👤 ผู้ใช้งานที่ลงทะเบียนแล้ว

ผู้ลงทะเบียนใหม่ทุกคนตั้งรหัสผ่านของตัวเองตอนสมัคร และเริ่มต้นด้วยสถานะ "ยังไม่กำหนดบทบาท" — Admin ต้องกำหนดบทบาท สิทธิ์และข้อมูลผู้ใช้งานที่นี่

ชื่อ-สกุล ประเภท/ตำแหน่ง Username (เลขบัตรฯ) รหัสผ่าน บทบาท ที่อยู่ เบอร์โทร เชื่อมโยง CG ลงทะเบียนเมื่อ
